This page contains a synthetic division script in C. Synthetic division is a mathematical technique used to divide polynomials by a linear factor. The program takes the degree of the polynomial, the coefficients, and the divisor as input and calculates the results of synthetic division. 

#include <stdio.h>

void syntheticDivision(int coefficients[], int n, int divisor, int results[]) {
	int i;
	results[0] = coefficients[0];
	for (i = 1; i < n; i++) {
		results[i] = results[i - 1] * divisor + coefficients[i];
	}
}

int main() {
	int n;
	printf("Enter the degree of the polynomial: ");
	scanf("%d", &n);

	int coefficients[n + 1];
	int i;

	printf("Enter the coefficients of the polynomial from the highest degree to lowest:\n");
	for (i = n; i >= 0; i--) {
		printf("Coefficient for x^%d: ", i);
		scanf("%d", &coefficients[i]);
	}

	int divisor;
	printf("Enter the divisor (c): ");
	scanf("%d", &divisor);

	int results[n]; // Array to store the results of synthetic division

	syntheticDivision(coefficients, n + 1, divisor, results);

	printf("The results of synthetic division are:\n");
	for (i = 0; i < n; i++) {
		printf("x^%d: %d\n", n - i - 1, results[i]);
	}

	return 0;
}

The loop for synthetic division within the `syntheticDivision` function iteratively computes each term of the quotient polynomial. Beginning with an initialization of `i = 1`, the loop progresses through the coefficients of the polynomial, excluding the constant term, and employs the synthetic division algorithm to determine each term of the result polynomial. The update equation, `results[i] = results[i - 1] * divisor + coefficients[i]`, sums up the process by multiplying the previous result by the divisor and adding the next coefficient. The loop continues until `i` reaches the degree of the p olynomial plus one (`n`), ensuring that all coefficients of the quotient polynomial are computed.
