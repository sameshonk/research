Pointers in C are variables that store memory addresses, allowing indirect access to data. They are a powerful feature, often used to pass references to data structures, modify variables indirectly, and manage memory efficiently by dynamically allocating and deallocating memory. Pointers enable low-level memory manipulation and are a fundamental concept in C programming.

The follwing C program illustrates the concept of pointers, a fundamental feature in the C programming language. In this program, we have a main function and a helper function called increment. 

#include <stdio.h>

void increment(int *p)  // note that it accepts a pointer to an int
{
	*p = *p + 1;        // add one to the thing p points to
}

int main(void)
{
	int i = 10;
	int *j = &i;  // note the address-of; turns it into a pointer to i

	printf("i is %d\n", i);        // prints "10"
	printf("i is also %d\n", *j);  // prints "10"

	increment(j);                  // j is an int*--to i

	printf("i is %d\n", i);        // prints "11"!
}

In the main function, an integer variable 'i' is declared and initialized to 10. A pointer 'j' is also declared, and it is assigned the address of 'i' using the & operator. This means that 'j' now points to the memory location of 'i'.

The program proceeds to print the value of 'i' using printf, resulting in the output "i is 10." It also prints the value pointed to by 'j', which is the same as 'i', so it prints "i is also 10."

	printf("i is also %d\n", *j);

The next line calls the 'increment' function, passing 'j' as an argument. This function takes a pointer to an integer and increments the value it points to by 1.

Finally, after the 'increment' function call, the program prints the value of 'i' again using printf, which now prints "i is 11" because 'i' was modified by the 'increment' function.
